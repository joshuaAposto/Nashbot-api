API Documentation

Endpoint: /gpt3

Retrieve responses from NashBot based on prompts.

GET /nashbot3?prompt=<text>

Parameters

  • prompt (string): The prompt text to query NashBot.

Response Example

{
  "status": 200,
  "creator": "NashBot",
  "result": {
    "reply": "NashBot is an AI chatbot designed to provide intelligent responses and integrate with various services. It handles diverse queries, offers context-aware responses, and ensures data security and privacy."
  },
  "team": "Developed by Nash"
}
